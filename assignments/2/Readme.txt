How to run:
To run a specific type of the problem, use: 
python bigram.py *type*
Example: python bigram.py 1 or python bigram.py 2
1 - No smoothing
2 - Add one smoothing
3 - Good turing discount smoothing

To run all types, use:
python bigram.py

Input test string is hardcoded within the file as the variable "input_sentence".
